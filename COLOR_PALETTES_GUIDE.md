# 🎨 Professional Color Scheme - Authentication Pages

Your login, sign up, and forgot password pages now feature a **clean, professional corporate color scheme** designed for maximum trustworthiness and business credibility.

---

## 💼 Professional Color Palette

| Element | Color | Purpose |
|---------|-------|---------|
| **Outer Background** | `#f8fafc` | Light, neutral background for visual balance |
| **Left Panel** | `#1e3a5f` | Deep professional blue |
| **Right Panel** | `#2c5aa0` | Rich corporate blue |
| **Brand/Accent Color** | `#3b82f6` | Modern professional accent |
| **Input Fields** | `#1a324a` | Dark, secure input background |
| **Secondary Text** | `#94a3b8` | Professional gray for secondary content |

## 🔧 Color Customization

To modify the professional color scheme, update the `colors` dictionary in the `inject_auth_css()` function:

```python
colors = {
    "outer_bg": "#f8fafc",      # Light neutral background
    "left_bg": "#1e3a5f",       # Deep professional blue
    "right_bg": "#2c5aa0",      # Rich corporate blue
    "input_bg": "#1a324a",      # Dark input fields
    "brand_color": "#3b82f6",   # Professional blue accent
    "text_secondary": "#94a3b8" # Professional gray text
}
```
